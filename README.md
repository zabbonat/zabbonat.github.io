# zabbonat.github.io

Personal academic website of Diletta Abbonato.

The site offers two viewing modes:

- **Classic Mode** – A clean, traditional academic layout with sections for research tools, publications, teaching, and CV.
- **RPG Mode** – An alternative pixel-art inspired interface that presents the same content through an interactive, gamified experience.
